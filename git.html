Git

General Manual

  • git clone <-- clones remote repo into current directory
  • git status <-- tells you what is tracked and what is not
  • git add <-- lets you add files to be tracked
  • git rm <-- lets you remove files to be tracked
  • git commit <-- commits changes (use -m "description" to put a note on the changes)
  • git pull <-- pulls updates into local repo (must also be current directory)
  • git push <-- pushes the local repo to the remote repo (must be used after a commit, usually followed by origin master)

Common Usage

  • cd $HOME/Scripts/
  • git clone https://github/user/Git-Project.git
  • *edit files*
  • cd $HOME/Scripts/Git-Project/
  • git commit -m "Fixed bugs."
  • git push origin master
